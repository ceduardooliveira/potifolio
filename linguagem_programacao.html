Variáveis e Operadores.

Conversões entre tipos de variáveis e Input Java

Variáveis em Java armazenam dados e têm tipos específicos, como int, double e String. Operadores, como +, -, *, e /, permitem operações matemáticas e lógicas entre variáveis. Conversões entre tipos, como de int para double, podem ser feitas implicitamente ou explicitamente (casting). Input em Java geralmente usa a classe Scanner para ler dados do usuário via terminal. Um exemplo de leitura de input seria Scanner scanner = new Scanner(System.in); int numero = scanner.nextInt();.

Neste segundo exemplo, é um programa que mostra o que aprendi com recursividade, aqui estou implementando uma função para calcular o fatorial de um número inteiro N digitado pelo usuário.

Neste código, a função calcularFatorial é implementada de forma recursiva para calcular o fatorial de um número inteiro. A função main solicita ao usuário que digite um número inteiro e, em seguida, chama a função calcularFatorial para calcular o fatorial desse número. Ao executar o programa, você poderá digitar um número inteiro. Por exemplo, se você digitar 5, o programa calculará o fatorial de 5, que é 5 * 4 * 3 * 2 * 1, resultando em 120. O programa então imprimirá "O fatorial de 5 é: 120".

A função calcularFatorial é definida de forma recursiva da seguinte maneira: Se o número for 0 ou 1, o fatorial é 1. Caso contrário, o fatorial de um número n é n multiplicado pelo fatorial de n-1.

No codigo ao lado, construi um sistema simples de gerenciamento de contas bancárias. Classe ContaBancaria A classe ContaBancaria representa uma conta bancária e possui três atributos privados: numero: um inteiro que representa o número da conta. saldo: um valor em ponto flutuante que representa o saldo atual da conta. titular: uma string que representa o nome do titular da conta. O construtor inicializa os atributos da conta bancária com os valores fornecidos quando uma instância da classe é criada. O método depositar adiciona um valor ao saldo da conta e imprime uma mensagem confirmando o depósito. O método sacar verifica se há saldo suficiente na conta para realizar o saque. Se houver, subtrai o valor do saldo e imprime uma mensagem confirmando o saque. Caso contrário, imprime uma mensagem indicando saldo insuficiente. O metódo vericarSaldo retorna o saldo atual da conta.

A classe Main, contém o metódo main, que é o ponto de entrada do programa. No metódo main são realizadas as seguintes operações:

    Criação de uma instância da classe ContaBancaria.

O método depositar é chamado para adicionar 500.0 ao saldo, e o saldo atual é exibido no console. O saldo após esta operação será 1500.0.

    Deposito de 500.00 na conta.
    Saque de 200.0 da conta.

O método sacar é chamado para subtrair 200.0 do saldo, e o saldo atual é exibido no console. O saldo após esta operação será 1300.0.

    Tentativa de saque de 1500.0 da conta.

O método sacar é chamado para tentar subtrair 1500.0 do saldo. Como o saldo é insuficiente (apenas 1300.0), uma mensagem de saldo insuficiente é exibida e o saldo permanece inalterado.
