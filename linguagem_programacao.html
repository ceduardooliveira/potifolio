Conversões entre tipos de variáveis e Input Java

Variáveis em Java armazenam dados e têm tipos específicos, como int, double e String. Operadores, como +, -, *, e /, permitem operações matemáticas e lógicas entre variáveis. Conversões entre tipos, como de int para double, podem ser feitas implicitamente ou explicitamente (casting). Input em Java geralmente usa a classe Scanner para ler dados do usuário via terminal. Um exemplo de leitura de input seria Scanner scanner = new Scanner(System.in); int numero = scanner.nextInt();.

Calculando a área do Triângulo Retângulo

Estrutura de decisão (if/else)

No código ao lado é um sisteminha onde o usário tem que calcular a média do aluno. O usuário tem que inserir duas notas para calcular, se a nota for menor que 50, deve printar na tela aluno "REPROVADO", caso contrário "APROVADO".
